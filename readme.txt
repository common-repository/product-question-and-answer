This plugin allows the customers to ask questions about each product in your store and any one can answers for those Questions.

== Description ==

Features:-
1) Let your users ask a question on specific WooCommerce
products.
2) Let your users answer a question on the product page.
3) Manage easily questions and answers.
4) This plugin allows register customer of your shop to ask
questions about products. Both admin and other register
customer can answers to them and guest customer can see all
question and answer of all product of your shop.
5) Work with simple product and variable products.

Avails logged-in customers to ask question and give answer about
each product.
User can give multiple answers for one question for same login
account.
It is working on simple product and variable product in woocommerce.

Admin can enable Product Q/A tab in particular product page and
disable for particular product page.
Admin can show list of question order by post date ,user name ,user
email id and approve status.

No configuration needed! just install and it will start working from
there, you can change the settings at any time though.
